# Installation

## Stable release

To install ModelTrain_FR, run this command in your terminal:

```sh
uv add ModelStudy_FR
```

Or if you prefer to use `pip`:

```sh
pip install ModelStudy_FR
```

## From source

The source files for ModelTrain_FR can be downloaded from the [Github repo](https://github.com/sharatkatz/ModelStudy_FR).

You can either clone the public repository:

```sh
git clone git://github.com/sharatkatz/ModelStudy_FR
```

Or download the [tarball](https://github.com/sharatkatz/ModelStudy_FR/tarball/master):

```sh
curl -OJL https://github.com/sharatkatz/ModelStudy_FR/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd ModelStudy_FR
uv pip install .
```
